# Editor Revert Plan

The user wants the Editor to be simpler - just preview, not split-view with code editor.

## Current Issues:
1. Split-view with code editor looks "clumsy"
2. Code editor should only be in Generate page (AI builder)
3. Editor should just show preview + chat toggle

## Solution:
Keep the Editor simple with:
- Full-width preview (no code editor)
- Chat button to toggle AI chat overlay
- All tools in top bar
- Clean, simple interface

This matches the original design better and is less cluttered.
